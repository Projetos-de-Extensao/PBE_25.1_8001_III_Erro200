Fluxo de Status dos Pedidos

O sistema possui um fluxo bem definido de status para cada pedido, garantindo o rastreamento correto da entrega em cada etapa.

Estados Possíveis

  1. A_CONFIRMAR
  2. Estado inicial quando o pedido é criado

  3. O pedido está disponível para aceitação por um entregador

  4. ACEITO

  5. O entregador aceitou o pedido

  6. Aguardando início do transporte

  7. EM_TRANSITO

  8. Entregador está transportando o pedido até o porto

  9. Iniciado após o entregador confirmar o início do transporte

  10. NO_PORTO

  11. Pedido foi entregue ao porto

  12. Disponível para aceitação por um barqueiro

  13. EM_TRAVESSIA

  14. Barqueiro está realizando a travessia do pedido

  15. Iniciado após o barqueiro aceitar a travessia

  16. NO_POSTO

  17. Pedido chegou ao posto de destino

  18. Aguardando retirada pelo cliente

  19. ENTREGUE

  20. Cliente recebeu o pedido

  21. Aguardando confirmação final do cliente

  22. CONCLUIDO

  23. Estado final do pedido
  24. Cliente confirmou o recebimento

Transições de Status

Ações do Entregador

  • A_CONFIRMAR → ACEITO (via aceitar_entrega)
  • ACEITO → EM_TRANSITO (via iniciar_transporte)
  • EM_TRANSITO → NO_PORTO (via entregar_no_porto)

Ações do Barqueiro

  • NO_PORTO → EM_TRAVESSIA (via aceitar_travessia)
  • EM_TRAVESSIA → ENTREGUE (via marcar_entregue)

Ações do Cliente

  • ENTREGUE → CONCLUIDO (via marcar_concluido)

Regras de Negócio

  1. Restrições de Usuário
  2. Apenas entregadores podem aceitar e transportar pedidos
  3. Apenas barqueiros podem realizar travessias

  4. Apenas o cliente que criou o pedido pode marcá-lo como concluído

  5. Ordem dos Status

  6. Os status devem seguir a ordem definida
  7. Não é possível pular etapas

  8. Um pedido não pode voltar a um status anterior

  9. Visibilidade

  10. Entregadores veem pedidos em A_CONFIRMAR e seus próprios pedidos aceitos
  11. Barqueiros veem pedidos em NO_PORTO e seus próprios pedidos em travessia
  12. Clientes veem apenas seus próprios pedidos
  13. Administradores veem todos os pedidos
